# Para jogar

- Inicialize o servidor:
    ```
    python3 servidor.py
    ```

- Após a inicialização do servidor, rode o cliente em dois terminais diferentes:
    ```
    python3 cliente.py
    ```


*É necessário 2 jogadores registrados para a partida começar
*Comunicação tcp/ip
